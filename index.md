# 프로젝트 기술 표준 가이드

1. 개발 환경 및 기술 스택 (Tech Stack)
  - Language: Java 21 (Record, Switch Pattern Matching, Virtual Threads 적극 활용)
  - Framework: Spring Boot 4.x
  - Persistence: Spring Data JPA & MyBatis 3.x
  - Build Tool: Gradle
  - Database: PostgreSQL 및 커넥션 풀 ([HikariCP](appendix/hikariCP.md)) 설정 표준
