# Persistence Layer 표준 (Hybrid: JPA + MyBatis)

- 기술 선택 기준
    - 단일 엘티티 CRUD 및 객체 중심 로직 → JPA
    - 복잡한 대량 통계 쿼리, Projections, 특정 DB 종속 쿼리 

- JPA 표준
    - 엔티티 설계: 가급적 Long 타입의 Sequence PK 사용

- MyBatis 표준
    - Mapper XML 위치 및 Naming (Interface와 매칭 규칙)
    - _paramter 사용 금지 및 @Param 명시
    - CamelCase 자동 매핑 설정 및 TypeHandler 활용
